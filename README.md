# Retail Sales & Returns Analysis | Tableau Prep + Tableau Public

## Project Overview
Developed an end-to-end data cleaning and transformation workflow for a multi-region e-commerce retail dataset using Tableau Prep. Processed and standardized data from multiple order sources (Central, East, West, and South regions) to improve reporting accuracy and business insights.

## Key Highlights
- Built calculated fields for missing regional data and created proper order/ship date fields using MAKEDATE()
- Cleaned and transformed inconsistent datasets by handling null values, removing duplicates, correcting data types, and standardizing formats
- Processed sales and discount fields by converting string-based currency values into decimal numeric formats
- Performed data union and schema alignment across multiple regional order datasets
- Integrated return reasons data using joins and created return status indicators for business analysis
- Applied text cleaning techniques including whitespace removal, title casing, and approver name extraction/grouping
- Engineered new analytical fields such as Order to Shipment Days to track operational efficiency
- Exported the final cleaned dataset in .xlsx format for visualization and dashboard creation in Tableau Public
